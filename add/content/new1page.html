Элемент meta определяет метаданные документа
Чтобы документ корректно отображал текст, необходимо задать кодировку с помощью атрибута charset
Рекомендуемой кодировкой является utf-8

Элемент meta также имеет два атрибута: name и content. Атрибут name содержит имя метаданных, а content - их значение
По умолчанию в HTML определены пять типов метаданных
		Элемент details позволяет создавать раскрываемый блок, который по умолчанию скрыт.
		Данный элемент содержит элемент summary, который представляет заголовок для блока,
